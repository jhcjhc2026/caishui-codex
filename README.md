# 财税服务内容资产库

这个仓库用于管理财税服务相关的内容资产、选题研究、短视频文案、公众号参考资料、客户拓客表格和日常工作流。

## 目录说明

- `01-每日内容生产包/`：每日财税自媒体内容包。
- `02-公众号文章与参考资料/`：公众号文章、Word 参考稿、财税资料。
- `03-短视频选题与抖音数据/`：抖音低粉爆款选题、搜索结果、视频详情和趋势数据。
- `04-标题简介与客户群素材/`：短视频标题、作品简介、话题标签、客户群话术素材。
- `05-图片素材/`：公众号头像、朋友圈配图、候选封面截图等图片。
- `06-客户拓客与表格/`：客户线索、电话拓客记录表、Excel 表格。
- `07-工作流SOP/`：固定工作流、操作说明、内容生产规则。
- `08-工具脚本/`：辅助处理表格、数据或内容的脚本。
- `99-本地临时素材/`：录屏、截图、浏览器临时文件等，不建议上传 GitHub。

## 使用原则

1. 客户信息、手机号、合同、金额、税务细节等敏感内容，只建议放在私有仓库。
2. 视频录屏、浏览器缓存、登录资料不进入 GitHub。
3. 每天新增内容包按日期命名，便于回溯。
4. 抖音选题数据保留原始数据，方便复盘爆款来源。
